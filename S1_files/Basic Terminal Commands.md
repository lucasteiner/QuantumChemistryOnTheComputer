### Commands:
Most of the time, you will need **ls**, **cp** and **gedit**.
As a further tip: use Arrows 🔼 to get your last command.

**ls** (list): shows all files of your current directory
```sh
ls
```

**mkdir** (make directory): Creates a new directory
```sh
mkdir qc_versuchs
```

**cd** (change directory): Changes current directory
```sh
cd qc_versuchs
```

**gedit** (text editor): opens a file with the gedit-texteditor. 
```sh
gedit co.inp
``` 
With this command you can
- open and create new files
- open and change old files
