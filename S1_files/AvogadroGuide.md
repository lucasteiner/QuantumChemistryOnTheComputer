### Display a Molecule
In the terminal, enter the command
```sh
avogadro2
```

At the top: Click /File/Open
Select *co.out*
If not displayed, check i**Display Types/ Ball and Stick**
![[Pasted image 20241027191515.png]]
### Display a Molecular Orbital
At the top: Click /Analysis/Create Surfaces
Select **Surface: Molecular Orbital**
Select one MO
Click Calculate
If not displayed, check **Display Types/ Meshes**

![[Pasted image 20240422201940.png]]


### Make a Picture
**Click /File/Export/Graphics...** 
Select your calculation directory
and save as **filename.png** (e.g. sigma.png, pi.png, pi_star.png)
